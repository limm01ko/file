## Descrição
Este projeto é um módulo em Node.js que permite criar e ler arquivos de forma simples.

## Funcionalidades
- Criar arquivos
- Ler arquivos
- Suporte a caminhos relativos e absolutos
- Suporte a múltiplos formatos (txt, json, xml, etc)
